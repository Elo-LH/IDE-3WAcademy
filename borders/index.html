Les bordures en CSS

C'est la propriété border qui permet d'appliquer une bordure à un élément. C'est en fait une propriété raccourcie qui englobe 3 propriétés :

  • border-color pour la couleur
  • border-width pour son épaisseur (valeur en pixels)
  • border-style pour le style
